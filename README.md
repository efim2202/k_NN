# Метод k_NN ближайших соседей

Задача: классифицировать тестовые точки исходя из данных двух множеств, образуемых двумя типами точек.

Что используем: для классификации нам необходима матрица расстояний от каждой тестовой точки до всех исходных,
ее заполняем добавляя к матрицам тестовых и исходных точек новую ось так, чтобы по правилам broadcasting мы могли найти сумму модуля разности координат (Манхэттенское расстояние), получив расстояние от тестовой до исходной точки, так мы заполняем всю матрицу расстояний, далее анализируя матрицу расстояний мы относим каждую тестовую точку к определенному множеству.

Результат: для небольших данныых получилось классифицировать точки относительно заданных множеств, использовали полную векторизацию, составляя матрицу расстояний, а, значит, данный код будет хорошо себя чувствовать и для больших данных, как это реализовать у меня ума не хватило)
